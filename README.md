# Visual Macro MVP

비주얼 매크로 편집기(MVP): 템플릿 이미지와 감시 영역을 설정하고, 화면에서 동일 이미지를 좌상단부터 순차 클릭합니다.

## 1) 설치
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## 2) macOS 권한 설정
- 시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용: 터미널(or Python) 허용
- 시스템 설정 > 개인정보 보호 및 보안 > 화면 기록: 터미널(or Python) 허용
- 권한 변경 후 앱을 재시작하세요.
- 참고: 마우스 이동이 동작하지 않으면 손쉬운 사용 권한을 다시 토글한 뒤 터미널/파이썬을 재실행하세요. PyAutoGUI FAILSAFE는 비활성화되어 있습니다.

## 3) 실행
```bash
python main.py
```

## 4) 사용 방법
- 템플릿 캡처: 버튼을 눌러 화면 영역을 드래그 → PNG로 저장 → 1번 스텝에 자동 반영
- 영역 지정: 버튼을 눌러 감시 영역을 드래그(미지정 시 전체 화면)
- 자동화 시작: 동일 템플릿을 화면에서 모두 탐색 → 좌상단부터 미방문 매치를 순차 클릭
- 중지: 중지 버튼

## 5) 팁
- 탐지 성능: 감시 영역을 좁힐수록 빠르고 정확합니다.
- 템플릿: 경계가 뚜렷한 작은 영역을 권장합니다.
- 멀티 디스플레이: 현재 활성 디스플레이에 대해 캡처/탐지를 수행합니다.

## 6) 문제 해결
- 클릭이 동작하지 않음: 손쉬운 사용/화면 기록 권한을 확인하세요.
- 매치가 잡히지 않음: 템플릿을 더 작게/명확하게, 밝기/해상도 차이가 없는지 확인하세요.
- 너무 느림: 감시 영역 축소, 템플릿 단순화, 해상도 낮추기.
